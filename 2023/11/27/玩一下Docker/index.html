玩一下Docker

Created2023-11-27|Updated2023-11-27|分类
|Post Views:

1、获得镜像

获取已经制作好的镜像,相当于一个带完整环境的安装包

拉取镜像: pull mysql
查看镜像: images
删除镜像: rmi mysql

2、操作容器

容器相当于一台小的虚拟机,使用容器安装一个镜像包

创建容器: create
创建容器 ,指定名字(–name=名字) ,指定环境变量(-e 制作者指定),指定挂载在本机的目录(-v 本机目录:虚拟机目录),指定端口(-p) ; 例子如下

  • 启停容器: start 、stop、 restart
  • 删除容器: rm mysql
  • 进入容器: exec -it mysql bash
  • 退出容器: exit
  • 查看容器: ps -a

3、宿主机挂载

容器内文件操作麻烦,挂载在宿主机上可方便操作

挂载数据卷: 在创建时-v 数据卷名称或目录地址:容器内目录

  • 创建 volume create
  • 查看 volume ls
  • 删除 volume rm
  • 查看详情 volume inspect

4、制作镜像

制作自己的镜像供它人使用,指令存Dockerfile中

  • 指定基础镜像 : FROM
  • 环境变量 : ENV
  • 拷贝本地文件到镜像目录 : COPY ./ ./
  • 执行需要的shell命令 : RUN
  • 指定端口 : EXPOSE
  • 设定启动命令 : ENTRYPOINT
